# 問い合わせ管理アプリ

## 📌 概要
Laravelで作成した問い合わせフォーム管理アプリです。  
ユーザーからの問い合わせを登録・一覧表示・編集・削除できるCRUD機能に加え、検索機能やページネーションも実装しています。

---

## 🛠 機能
- 問い合わせ登録
- 一覧表示
- 詳細表示
- 編集・削除
- バリデーション（FormRequest使用）
- 🔍 検索機能
- 📄 ページネーション

---

## 💻 使用技術
- PHP（Laravel）
- MySQL
- Tailwind CSS

---

## ✨ 工夫した点
- FormRequestを使用してバリデーション処理を分離
- Serviceクラスを作成し、ビジネスロジックを整理
- 検索機能を実装し、ユーザビリティを向上
- ページネーションを導入し、大量データでも見やすく設計
- MVC構造を意識した設計
