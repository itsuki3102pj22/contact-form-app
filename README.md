お問い合わせ管理アプリ（Contact Management App）

📌 概要

Laravelを使用して開発したお問い合わせ管理アプリです。
ユーザーからの問い合わせ情報を登録・一覧表示・検索・詳細確認・編集・削除できるCRUD機能を実装しています。

⸻

🔧 使用技術
	•	PHP（Laravel）
	•	Blade（テンプレートエンジン）
	•	MySQL
	•	Tailwind CSS

⸻

✨ 主な機能
	•	📋 一覧表示（ページネーション対応）
	•	🔍 検索機能（名前の部分一致検索・複数キーワード対応）
	•	➕ 新規登録
	•	🔎 詳細表示
	•	✏️ 編集機能（バリデーション対応）
	•	🗑️ 削除機能（確認ダイアログ付き）

⸻

🧠 工夫したポイント
	•	検索機能の強化
	•	スペース区切りによる複数キーワード検索に対応
	•	入力保持（UX改善）
	•	old()を使用して、バリデーションエラー時の入力内容保持を実装
	•	条件分岐の活用
	•	URLが存在する場合のみリンクとして表示
	•	誤操作防止
	•	削除時にJavaScriptの確認ダイアログを表示
	•	RESTful設計
	•	PUT / DELETE メソッドを使用した適切なルーティング
